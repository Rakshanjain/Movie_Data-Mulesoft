# Movie_Database-Mulesoft
### 1. Created database with table name "MOVIES" and stored interesting movie names with the names of lead actor, actress, year of release and the director name. 
### 2. File 'movies.sql' contains SQL queries for creating table, inserting values into table and queries.
### 3. File 'database.php' is used to connect XAMPP server to our project.
### 4. File 'index.php' is used to include database connection in our project.
